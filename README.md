# 🛍️ ShopNest — E-Commerce Frontend

## 🎯 Objective
- Build a fully responsive e-commerce website using only frontend technologies
- Simulate a complete shopping flow: browse → product detail → cart → checkout
- Practice DOM manipulation, state management, and form validation without a backend

## 🛠️ Tools & Technologies
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- LocalStorage (for cart persistence)
- Font Awesome (icons)
- Google Fonts (Poppins & Inter)
- VS Code + Live Server

## ✨ Functionalities
- Responsive navbar with mobile hamburger menu
- Product listing with category filters and search
- Hot deals section with live countdown timers
- Dynamic product detail page (image gallery, color/size selector, reviews)
- Add to cart, buy now, and wishlist actions
- Cart with quantity control, coupon codes, and live price calculation
- Multi-step checkout with shipping & payment options
- Form validation with real-time error feedback
- Order confirmation with generated order ID
- Cart data persists across all pages via localStorage

## 🔮 Future Improvements
- User authentication (login/signup)
- Backend integration (Node.js / Firebase)
- Real payment gateway integration
